# 硬件开发手册：TPS5450 降压电源模块 (24V -> 5V)

> **设计目标**：为机器人控制系统提供稳定的 5V/5A 动力源。

---

## 1. 核心性能与应用场景

### 1.1 电流承载能力
- **额定输出电流**：5A（连续工作）。
- **峰值输出电流**：6A（短时峰值）。

### 1.2 典型应用场景
- **机器人动力电源**：将 6S 锂电池（额定 22.2V，满电 25.2V）降压至 5V。
- **高功率舵机驱动**：提供足够的瞬间电流以支撑舵机堵转时的负荷。

---

## 2. FB (Feedback) 引脚控制逻辑详解

FB 引脚是 DCDC 芯片的“眼睛”，它决定了输出电压能否稳定在 5V。

### 2.1 工作原理
TPS5450 内部包含一个精度极高的 **1.221V 基准电压源** 和一个 **误差放大器**。
1. **采样**：通过 R1 和 R2 组成的分压电路，将输出电压 $V_{out}$ 进行比例缩小，接入 FB 引脚。
2. **比较**：芯片内部实时比较 $V_{FB}$ 与 $1.221V$ 基准。
3. **调整**：
    - 如果 $V_{FB} < 1.221V$，说明输出电压偏低，芯片会自动**增加 PWM 占空比**，让 MOS 管导通时间变长，从而提升输出。
    - 如果 $V_{FB} > 1.221V$，则**减小占空比**，降低输出。



### 2.2 阻值计算
为了得到 5V 输出，我们需要按照以下公式配置分压电阻：

$$V_{out} = 1.221 \times (1 + \frac{R_1}{R_2})$$

- **推荐选型**：$R_1 = 10k\Omega$，通过计算得出 $R_2 \approx 3.24k\Omega$。

## 3. 关键元器件：电感 (Inductor) 选型

电感是 Buck 电路中储能和滤波的核心，选型不当会导致效率低下、纹波过大甚至芯片烧毁。

### 3.1 电感量计算公式
电感的选择平衡了电流纹波和瞬态响应。通常取电流纹波系数 $K_{ind} = 0.2 \sim 0.3$：

$$L = \frac{V_{in(max)} - V_{out}}{I_{out} \times K_{ind}} \times \frac{V_{out}}{V_{in(max)} \times f_{sw}}$$

其中：
- $V_{in(max)} = 24V$
- $f_{sw} = 500kHz$ (TPS5450 的固定频率)
- **计算结论**：推荐选用 **$15\mu H$ 到 $22\mu H$** 的屏蔽功率电感。



### 3.2 选型三大核心指标
在购买电感时，必须核对以下参数：
1. **额定电流 (Irms)**：必须大于你的最大工作电流（5A），建议选 **7A** 以上。
2. **饱和电流 (Isat)**：电感在高电流下不能“磁饱和”。公式：$I_{sat} > I_{out} + \frac{\Delta I_L}{2}$。建议选 **8A-10A**。
3. **直流电阻 (DCR)**：DCR 越小，热损耗越低。对于 5A 应用，DCR 应控制在 **20mΩ 以下**。

## 4. 调试检查清单
- [ ] 检查 FB 引脚焊点是否虚焊（FB 悬空会导致电压飞升至输入电压，烧毁后端设备！）

### 5.4 特别警告：SW 节点的“过孔”

在 PCB 布局时，SW（开关）引脚的走线处理直接决定了芯片的寿命。

#### 1. 为什么 SW 处过孔多会导致“炸板”？
SW 节点是一个高频切换节点（在 TPS5450 中为 500kHz），它具有极高的 $dv/dt$（电压变化率）和 $di/dt$（电流变化率）。

- **寄生电感效应**：每一个过孔都会引入约 **1nH** 的寄生电感。
- **电压尖峰公式**：根据公式 $V_{spike} = L \times \frac{di}{dt}$，在高频开关瞬间，过孔带来的电感 $L$ 会与电流变化产生巨大的感应电动势，形成超高压尖峰。
- **击穿机制**：如果 SW 处的尖峰电压超过了 TPS5450 的内部 MOSFET 耐压值（36V 额定，但尖峰可能瞬时突破 40V+），芯片内部的功率管会瞬间发生不可逆的物理击穿，导致“炸板”。



#### 2. SW 节点的布局原则
为了保证 24V 转 5V 的系统安全，请遵循以下布局守则：

- **单层优先**：尽量将 SW 引脚、电感、续流二极管放在 **同一层（通常是 Top 层）**。严禁为了布线方便，通过大量过孔将 SW 信号引到背面再接电感。
- **缩短路径**：SW 节点的走线要尽可能“短、粗、直”，减小环路面积。
- **严格控制过孔**：如果必须通过过孔散热，过孔的数量应经过严格计算，且不能直接打在高速电流路径上。
- **天线效应**：SW 节点本质上是一个辐射天线。过孔越多，噪声越容易通过过孔耦合到地平面或信号平面，造成严重的 EMI（电磁干扰）!

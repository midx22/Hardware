# 硬件设计专项手册：TVS vs. 稳压二极管 (Zener)

> **引言**：虽然它们在电路符号上都有个“钩子”，外观也极其相似，但在电路中扮演的角色截然不同——一个是应对突发灾难的“泄洪闸”，一个是维持恒定水位的“节流阀”。

---

## 1. TVS 瞬态电压抑制二极管 (The Protector)

TVS 专门设计用于吸收瞬间爆发的高能量（如静电 ESD、电机反电动势、雷击浪涌）。它的核心使命是**保命**。

### 1.1 核心工作逻辑
TVS 平时处于高阻态（像断路一样），对电路无影响。当两端电压超过阈值时，它会以 **ps（皮秒）** 级的速度瞬间导通，将电流泄放到地，并将电压钳位在一个安全范围内。

### 1.2 关键选型参数
- **$V_{RWM}$ (截止电压)**：正常工作时，电路电压不能超过这个值，否则 TVS 会误动作。
- **$V_C$ (钳位电压)**：**最核心指标**。在遭受大浪涌时，TVS 两端的最高电压。必须确保 $V_C$ 小于你后端芯片（如 MCU 或 LDO）的耐压极限。
- **$C_j$ (寄生电容)**：在高速信号线（CAN、USB）上，必须选 pF 级的低电容版本，否则信号会变形。



### 1.3 布局“军规”
- **接口最前线**：必须放在接插件入口，信号进板子的第一站。
- **严禁分支连接**：信号必须“穿过”TVS 的焊盘，不要指望静电会顺着分支出去寻找保护。

---

## 2. 稳压二极管 (The Standard)

稳压管（齐纳二极管）利用其反向击穿特性，在电流大范围变化时保持电压恒定。它的核心使命是**定标**。

### 2.1 核心工作逻辑
稳压管工作在持续的击穿状态。它通常配合一个**限流电阻**使用，为电路提供一个稳定的参考电压（基准电压）。

### 2.2 核心设计公式
必须串联电阻 $R_s$，计算公式为：
$$R_s = \frac{V_{in} - V_z}{I_z + I_{load}}$$
*其中 $I_z$ 建议选 **5mA - 20mA**，确保其稳定工作在击穿区。*



### 2.3 局限性
- **响应慢**：无法应对纳秒级的静电冲击。
- **功率小**：通常只用于参考电压或信号钳位，不能作为电源给电机等大功率设备供电。

---

## 3. 深度对比：一眼分清

| 特性指标 | TVS 二极管 (瞬态抑制) | 稳压二极管 (齐纳) |
| :--- | :--- | :--- |
| **主要功能** | **过压保护** (防静电、防浪涌) | **参考电压** (稳压、基准) |
| **工作时长** | 瞬态（毫秒/微秒级动作） | 持续（只要通电就一直工作） |
| **响应速度** | **极快** (ps/ns 级别) | 较慢 (us 级别) |
| **耐受能量** | **巨大** (瞬间可承受数千瓦) | 很小 (通常 0.5W - 5W) |
| **电路接法** | 并联在接口/地之间 | 并联 (必须配上限流电阻) |
| **失效表现** | 倾向于**短路失效** (牺牲自我保护全家) | 通常为过热烧毁或开路 |



---

## 4. 杂谈：工程避坑指南 💡

### 4.1 能不能互换？
* **把稳压管当 TVS 用？** 绝对不行。浪涌电流通常高达几十安培，稳压管那薄弱的身躯会瞬间汽化，保护不了任何东西。
* **把 TVS 当稳压管用？** 也不建议。TVS 的稳压精度很差，且电压会随电流剧烈跳动，无法提供准确的参考电压。

### 4.2 双向与单向怎么选？
* **直流电源输入**：选单向 (Uni-directional)。
* **通信线 (CAN/RS485)**：必须选双向 (Bi-directional)，因为信号线上可能存在正负波动的共模干扰。

### 4.3 散热与封装
对于 **Titan-120** 这种 24V 大功率系统：
- 电源入口推荐 **SMBJ30A** 或 **SMCJ30A** 封装，它们的“肉厚”，抗浪涌冲击能力强。
- 稳压管如果发烫，通常说明限流电阻给小了，或者输入压差过大。

---

## 5. 硬件调试 Checklist
- [ ] **TVS** 是否紧贴接插件摆放？
- [ ] **稳压管** 是否串联了限流电阻？（没加电阻通电必烧）
- [ ] 信号线上的 **TVS 电容** 是否影响了波形质量？（示波器观察上升沿）
- [ ] **$V_{RWM}$** 是否大于系统最大可能出现的正常电压？
